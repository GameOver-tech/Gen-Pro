2. KEY CONCEPTS LEARNED
2.1 Artificial Intelligence (AI)

AI is the simulation of human intelligence in machines that can:

Think
Learn
Solve problems
Make decisions
2.2 Machine Learning (ML)

Machine Learning is a subset of AI where machines learn patterns from data without being explicitly programmed.

Example:

Spam email detection
Recommendation systems (YouTube, Netflix)
2.3 Deep Learning (DL)

Deep Learning uses neural networks with multiple layers to process complex data like:

Images
Text
Speech
2.4 Generative AI

Generative AI is a branch of AI that can:

Generate text (ChatGPT)
Create images (Midjourney, DALL·E)
Generate code
Create audio/video
2.5 Large Language Models (LLMs)

LLMs like GPT work by:

Predicting the next word in a sequence
Using massive datasets
Learning patterns in language
